Design Questions

  1. Explain how design and development intersect. What role do you see a developer playing in the design process?
    Web designer creates visual element and layout of the website and web developer turns that website design into a fully functioning website. Now, I feel a developer can provide design inputs as he/she goes about building the website. It could be suggestions to use a newer/better technology that could make the same design more relevant in the long run or suggesting modifications that could improve overall performance. The function of a developer is to choose appropriate scripts and technologies to make the design provided by the web designer works.
  2. Describe how grids are useful, and how you generally go about using them, or implementing them.
    Grids are very useful when you want your design to look more appealing. There are some fundamental design rules when it comes to implementing a grid. If the object or content is placed at the intersection of grid lines, the design will be more visually appealing. In other words, grid is like a framework in which designer can put different display elements in a rational but attractive way. My own website features a number of photographs and I like to use the grid principle in my photography as well!
  3. Talk a little bit about pattern libraries and visual systems. What are they and how are they useful? Have you used them before?
    Pattern library is a collection of user interface design patterns or a collection of different visual elements like slide shows, navigations, new listing, related links etc. which appear multiple times in a website.
    Visual system sets the standards to the following elements of websites: typography, color, imagery, formats, logo and layouts. Designer has to set and follow some specific rules when it comes to visual systems.
    I have never used pattern libraries before but I hope to learn it and use it in my future projects.

  7. Explain how HTML, CSS, and JS interact - what is each layer responsible for?
    HTML is a mark up language used for structuring the contents of a web page, so it provides structure. CSS is used for visual styles for every content of your web page, it provides styling and Java script is a client side scripting language which interacts with HTML structure and content, that means it is used for interactive functionality in response to a click or event. These technologies combine to provide a seamless user experience so the user can easily interact with content presented on the webpage.
  8. Progressive enhancement or graceful degredation? Choose your side.
